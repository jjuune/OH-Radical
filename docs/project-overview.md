# 🌍 OH Radical 글로벌 브랜드 플랫폼 구축 기획안

> Human First, Earth Always

OH Radical 공식 웹사이트 구축 프로젝트는 단순한 기업 홈페이지 제작이 아닌,
차세대 친환경 정화 기술 브랜드로의 전환을 목표로 하는 글로벌 디지털 플랫폼 구축 프로젝트입니다.

---

# 📌 1. 프로젝트 배경

현재 대부분의 산업용/제조업 기반 웹사이트는 아래와 같은 한계를 가지고 있습니다.

| 기존 산업군 웹사이트 한계 | 문제점 |
|---|---|
| 제품 스펙 중심 구성 | 기술의 가치보다 장비 판매 느낌이 강함 |
| 텍스트 위주 설명 | 비전문가는 이해하기 어려움 |
| 브랜드 아이덴티티 부족 | 기억에 남지 않음 |
| 시각적 설득력 부족 | 첨단 기술 기업처럼 보이지 않음 |
| 글로벌 대응 부족 | 해외 바이어 설득력이 약함 |

OH Radical은 단순한 공기 살균 장비 제조사가 아니라:

> “보이지 않는 위험을 보이지 않게 해결하는 환경 기술 기업”

이라는 브랜드 방향성을 구축할 필요가 있습니다.

따라서 이번 프로젝트는:

- 글로벌 바이오테크 브랜드 포지셔닝
- ESG 친환경 기술 기업 이미지 강화
- B2G / B2B 신뢰도 확보
- 기술 스토리텔링 고도화
- 미래형 디지털 브랜드 플랫폼 구축

을 핵심 목표로 진행됩니다.

# ⚠️ 4. 기존 방식과의 차별화 전략

이번 프로젝트는 기존 산업용 홈페이지처럼:

- 제품 카탈로그 중심
- 기술 문서 중심
- 인증서 나열 중심
- 텍스트 설명 중심

으로 접근하지 않습니다.

OH Radical 플랫폼은:

> “문제를 해결하는 기술 경험 플랫폼”

방식으로 접근합니다.

---

# 🎬 7. 왜 3D / 모션이 중요한가?

OH Radical 기술은:

- 공기 흐름
- 바이러스 제거
- 입자 분해
- 분자 반응
- 정화 메커니즘

등 시각적으로 설명하기 어려운 요소가 핵심입니다.

따라서:

- 3D 모션
- 입자 애니메이션
- 데이터 시각화
- 인포그래픽
- 시네마틱 인터랙션

등이 매우 중요한 역할을 합니다.

이는 단순 “화려한 연출”이 아니라:

> “기술을 이해시키기 위한 UX 전략”

으로 접근합니다.
